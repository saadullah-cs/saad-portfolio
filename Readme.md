## 🚀 Overview
This repository contains the custom-engineered frontend architecture for the personal portfolio of **Saad Ullah**, Data Scientist and Founder of **NexGen Builds**. 

Moving beyond standard web layouts, this project implements a high-performance, interactive 3D environment designed to showcase advanced data pipelines, Python automation, and modern web solutions.

## 🧠 Core Technologies & Custom Implementations
While the foundational grid utilizes Bootstrap 5, the UI/UX has been heavily modified with custom scripting, dynamic routing, and advanced visual effects:

* **Interactive 3D WebGL Background:** Powered by `Vanta.js` (Globe), heavily customized with a pure black deep-space background (`#000000`) and NexGen Cyan (`#00FFFF`) neon nodes reacting to mouse/touch telemetry.
* **Dynamic Typography:** Integrated `Typed.js` for automated skill-cycling in the hero section.
* **Custom CSS Glassmorphism:** Engineered mobile-optimized frosted glass menu overlays and dynamic radial-gradient text-shadow arrays for maximum readability against complex 3D environments.
* **Asynchronous Data Filtering:** Utilizing `Isotope.js` for seamless, no-reload sorting of portfolio projects.
* **Zero-Lag Atmospheric Effects:** Pure CSS radial-gradient breathing animations and custom cursor trailing mechanics.

## 📂 Repository Structure
* `/assets/` - Contains all compiled CSS, custom JS modules, and optimized imagery.
* `index.html` - The master single-page application (SPA) layout.
* `404.html` - Custom error routing.
